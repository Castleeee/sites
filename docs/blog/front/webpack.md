## yarn和npm
node 自带npm  
`npm init (-y)` 初始化  
`npm i (-g) packages` 安装（全局）  
`npm update packages` 升级  
`npm uninstall packages` 卸载  
关联文件package.json  
  
`npm install yarn -g` 全局安装yarn  
  
`yarn add packages`:增加包  
`yarn init`:初始化yarn目录  
`yarn install`:安装所有包  
`yarn publish`:发布？？  
`yarn remove packages`:删除某个包  
关联文件package.json和yarn.lock
